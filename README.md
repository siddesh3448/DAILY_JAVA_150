# 🚀 Learning Approach

For every concept, I follow the same process:

1. Understand the theory.
2. Learn how it works internally.
3. Dry run the logic.
4. Implement it from scratch.
5. Solve multiple practice problems.
6. Review and improve the solution.
7. Commit the progress to GitHub.

The focus is on understanding rather than memorizing solutions.
